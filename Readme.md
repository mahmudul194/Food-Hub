# 🍔 Food Hub - Full Stack Food Delivery Web App

Food Hub is a modern and responsive full-stack food delivery web application that allows users to explore restaurants, browse menus, and order food seamlessly.

---

## 🌟 Live Preview

![Food Hub UI](./assets/preview.png)



---

## 🚀 Features

### 👤 User Features

* Secure User Authentication (Login/Register)
* Browse Restaurants & Menus
* Search Food Items
* Add to Cart & Place Orders
* Order Tracking & History
* Multiple Payment Options (Planned)

### 🛠️ Admin Features

* Admin Dashboard
* Manage Users
* Manage Food Items (CRUD)
* Manage Orders

---

## 🧑‍💻 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MySQL

## ⚙️ Installation Guide

### 1. Clone Repository

```bash
git clone https://github.com/mahmudul194/Food-Hub.git
cd Food-Hub
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Setup MySQL Database

* Create a database (e.g., `foodhub`)
* Import your SQL file
* Update DB credentials

### 4. Run Server

```bash
npm start
```

### 5. Open in Browser

```
http://localhost:3000
```

---

## 🔐 Environment Variables

Create `.env` file:

```
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=foodhub
JWT_SECRET=secretkey
```
